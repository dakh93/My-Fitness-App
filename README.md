# ❚█══█❚ My Fitness App ❚█══█❚

>[!NOTE]
>## **👉This is completely custom design and self idea of "Custom TKinter" fitness🏋️‍♀️ application to track your workouts during current month. The app contains the following features:**<br />

**➡ Login page, so it can be used by plenty of different users.**<br />
**➡ Register page**<br />
**➡ Custom Workout page where user can select which body parts to train and what difficulty**<br />
**➡ Completely Random Workout generation.**<br />
**➡ Calendar of the current month and which days were active ones with history of the saved workout of the day.**<br />
**➡ Stopwatch page where you can track you time**<br />

>[!NOTE]
**Project contains an existing database file(fitness.db) in it.**<br />
